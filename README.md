# MLFlow Quickstart

[Demo video](https://www.youtube.com/watch?v=cjeCAoW83_U)

## 1. Create venv

```bash
python -m venv venv
source venv/bin/activate
```

Install packages
```bash
pip install -r requirements.txt
```

## 2. Start MLFlow server

```bash
mlflow server --host localhost --port 5000
```
